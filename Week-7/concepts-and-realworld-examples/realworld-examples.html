Callback Approach

Traditional callback pattern - functions are passed to handle success or error.

// Callback-based approach
function fetchUsersCallback(successCallback, errorCallback) {
    fetch('https://jsonplaceholder.typicode.com/users')
        .then(response => response.json())
        .then(data => successCallback(data))
        .catch(error => errorCallback(error));
}

Promise Approach

Using Promises with .then() chaining for cleaner async flow.

// Promise-based approach
function fetchUsersPromise() {
    return fetch('https://jsonplaceholder.typicode.com/users')
        .then(response => response.json())
        .then(data => data)
        .catch(error => { throw error });
}

Async/Await Approach

Modern syntax that makes async code look synchronous and easier to read.

// Async/Await approach
async function fetchUsersAsync() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/users');
        const data = await response.json();
        return data;
    } catch (error) {
        throw error;
    }
}
